Columbia 1872 Map

Welcome to the Columbia 1872 Map Website

About

This application allows the user to view an interactive map of Columbia, SC as it was in 1872. The
user can tap on the map to see various buildings. The app allows an administrator to add and
delete buildings in the admin view. The text file with all the map layouts is saved and can
be backed-up through email in the Email Options view. This application was designed for the
Android platform. Due to the large size of the map that our application it is recommended that
this application is run on an Android tablet.

Features

General view and admin view maps
User editable locations

Location pop-ups in general map
- Pop-ups are created dynamically
Zooming and scrolling in general and admin views
